### Github limits

1. Generate new token and click `repo`: [Personal Access Tokens](https://github.com/settings/tokens)
2. Add token to your environment variables:
    ```bash
    echo 'export  GITHUB_TOKEN=your_github_token' >> ~/.bashrc
    source ~/.bashrc
    ```
